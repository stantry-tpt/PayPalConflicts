HypëêrTëêxt Mæärkúúp Læängúúæägëê (HTML) ïïs thëê mæäïïn mæärkúúp læängúúæägëê fõór wëêb pæägëês. HTML êélêémêénts äärêé thêé bääsíïc búúíïldíïng-blòõcks òõf wêébpäägêés

HTML ëëlëëmëënts fõórm thëë bùúìîldìîng blõócks õóf âàll wëëbsìîtëës.

Wëêb brõówsëêrs cåæn åælsõó rëêfëêr tõó Cåæscåædîíng Stylëê Shëêëêts (CSS) tõó dëêfîínëê thëê åæppëêåæråæncëê åænd låæyõóúút õóf tëêxt åænd õóthëêr måætëêrîíåæl. Thêê W3C, måãìîntåãìînêêr òôf bòôth thêê HTML åãnd thêê CSS ståãndåãrds, êêncòôúûråãgêês thêê úûsêê òôf CSS òôvêêr êêxplìîcìîtly prêêsêêntåãtìîòônåãl HTML måãrkúûp.
