HypèêrTèêxt Mæærkúúp Læængúúæægèê (HTML) îìs thèê mææîìn mæærkúúp læængúúæægèê fõór wèêb pæægèês. HTML éëléëméënts æåréë théë bæåsìîc büùìîldìîng-blöòcks öòf wéëbpæågéës

HTML èëlèëmèënts fóõrm thèë búüîíldîíng blóõcks óõf äãll wèëbsîítèës.

Wëèb bróôwsëèrs câän âälsóô rëèfëèr tóô Câäscâädîíng Stylëè Shëèëèts (CSS) tóô dëèfîínëè thëè âäppëèâärâäncëè âänd lâäyóôúüt óôf tëèxt âänd óôthëèr mâätëèrîíâäl. Thëë W3C, mààíîntààíînëër öóf böóth thëë HTML àànd thëë CSS stààndààrds, ëëncöóûûrààgëës thëë ûûsëë öóf CSS öóvëër ëëxplíîcíîtly prëësëëntààtíîöónààl HTML mààrkûûp.
